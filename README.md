#Git_Taek
1. Fyrir hvað stendur skammstöfunin RAM? Útskýrðu hvað RAM er.
		-Ram stendur fyrir Ramdom access memory. 
		-RAM er að nálgast fljótt af kerfum í tölvunni og hugbúnaði og geymir gögn og forrit ásamt forritum.

2. Fyrir hvað stendur skammstöfunin ROM? Útskýrðu hvað ROM er.
		-ROM stendur fyrir Read-Only Memory
		-ROM er notað til að "permenantly" geyma dót í tölvunni
